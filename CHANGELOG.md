## v1.9.1-rc.1

### 改进

- 发布校验支持 `v1.9.1-rc.1` 形式的 RC 标签，并在 GitHub Release 中自动标记为预发布。
- CI 分析流程改为使用锁文件获取依赖，并在分析前运行完整测试。
- SQLite 后台 isolate 操作改为独立打开数据库连接，减少跨 isolate 复用连接带来的稳定性风险。
- 数据导入流程增加替换前备份与失败回滚，并等待相关数据库管理器重新初始化完成。
- 应用数据保存改为排队写入，避免并发保存时的忙等和互相抢占。
- 历史记录异步写入改为 Future 队列，减少阅读进度频繁保存时的轮询等待。
- 历史记录批量刷新改为共享节流并发调度器，简化固定 worker 队列并保留批次节流。
- 关注更新批量检查复用节流并发调度器，减少固定 Channel worker 队列的重复实现。
- 共享节流并发调度器在最后一批任务后不再等待额外节流间隔。
- 收藏批量移动/复制改为事务提交后再通知，并在全文件夹删除提交后刷新计数。
- 收藏空批量移动、复制和删除会提前返回，避免无意义事务和监听通知。
- 本地漫画单项删除收敛为一次本地管理器通知，减少重复刷新。
- WebDAV 数据同步改为显式任务排队，减少上传/下载并发等待时的轮询。
- WebDAV 同步任务等待移除完成后的空延迟，减少排队任务切换时的无意义调度。
- 关闭窗口等待 WebDAV 上传时改为等待当前上传任务，减少退出前的固定间隔轮询。
- 关注更新后台检查改为等待 WebDAV 下载任务完成，避免固定间隔轮询和排队下载竞态。
- 漫画源数据保存改为合并排队写入，减少设置频繁变更时的重复等待。
- 漫画源设置解析与动态读取共用归一化逻辑，并统一包装 JS 回调设置项。
- 漫画源图片加载配置统一标准化，兼容 JS 返回的动态 Map 并拒绝非字符串键。
- 漫画源详情、章节图片和缩略图入口统一归一化 JS 返回值，减少动态 Map/List 类型边界问题。
- 漫画源详情入口补充嵌套字段归一化，兼容 JS 返回的动态推荐、评论和标签数据。
- 漫画源评论和章节评论入口统一归一化 JS 返回值，减少动态评论列表类型边界问题。
- 漫画源归档列表和归档下载地址入口统一校验 JS 返回值，减少归档下载类型边界问题。
- 漫画源搜索、分类和收藏列表入口统一归一化漫画条目，减少动态 Map 类型边界问题。
- 文件下载器改用写入队列串行落盘，减少多块并发下载时的忙等和缓冲累积。
- 缩略图加载并发限制改为显式许可队列，减少高并发加载时的固定间隔轮询。
- 图片响应回调改为统一执行并在异常路径释放，避免 JS 回调引用泄漏。
- 图片加载失败回调改为捕获后统一消费并释放，减少重试配置切换时的引用残留风险。
- 图片加载失败回调兼容 JS Map 返回的动态配置，并拒绝非字符串键配置。
- 图片加载失败重试按剩余预算判断，避免失败回调比预期多执行一次。
- 图片加载重试等待支持取消信号唤醒，减少取消后仍等待重试间隔的停顿。
- 缓存清理状态在异常时也会复位，避免后续清理任务被永久跳过。
- 缓存目录初始扫描失败时回退到空大小状态，避免后续缓存大小统计长期失效。
- 文件下载在任务创建阶段被停止时会关闭输出流和文件句柄，避免调用方等待挂起。
- 文件下载分块失败时会取消已启动分块并保留首个错误，避免错误路径被在途请求阻塞。
- 文件下载断点状态不足时不再继续发送完成状态，避免失败结果被误报。
- 文件下载进度定时器会在失败或提前结束时立即取消，避免关闭流后继续发送进度。
- 图片下载任务暂停或取消时会唤醒重试等待，减少退避延迟造成的暂停卡顿。
- 图片下载任务取消时会停止速度计时器，并在尚未创建下载路径时跳过本地库查询。
- 关注更新重试在最后一次失败后会立即返回，避免额外等待退避间隔。
- 历史记录刷新重试在最后一次失败后会立即返回，避免额外等待退避间隔。
- 阅读器自定义图片处理等待改为取消信号驱动，减少固定间隔轮询并传播异步错误。
- 图片脚本处理并发控制改为显式许可队列，减少高频处理时的轮询等待。
- JS 计算池初始化改为共享 Future，并用完成通知替代 isolate 任务关闭轮询。
- 更新日志页面改为按 Markdown 样式渲染，提升标题、列表和行内代码的阅读体验。

### 修复

- 修复 `Channel` 多个生产者等待同一释放信号时可能突破容量限制的问题。
- 修复文件下载分块请求失败时任务清理和错误传播不稳定的问题，并补充异常路径测试。
- 修复同路径网络请求 `prevent-parallel` 未覆盖完整请求生命周期的问题，并改为 Future 队列减少轮询等待。
- 修复 WebDAV 数据同步失败结果未统一刷新错误状态的问题，并补充异常路径测试。
- 修复共享图片下载取消时未终止底层图片流的问题。
- 修复共享图片流取消后立即重载同图可能复用已关闭流的问题。
- 修复图片下载任务暂停时未取消底层图片流且可能无法唤醒等待者的问题。
- 修复阅读器快速退出时最后阅读进度可能未写入的问题。
- 修复数据导入替换历史库前未等待异步历史写入完成的问题。
- 修复收藏管理器在独立初始化时可能等待未启动的应用数据初始化而挂起的问题。
- 修复收藏哈希刷新后台任务在测试清理临时数据库时可能产生 sqlite 竞态的问题。
- 修复 JS 引擎重置时旧实例未正确释放的问题。
- 修复网络内存缓存可能超过上限的问题，并补充缓存回归测试。
- 修复单条网络缓存超过内存上限时仍会写入缓存的问题。
- 替换 Flutter 已弃用的 `TickerMode.of` API。
